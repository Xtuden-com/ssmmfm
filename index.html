    Shortcut Help

    'p' chooses the first value

    the up and down arrows move between values

    after you have selected a value, 'l' shows you the value on the menu

    's' sets a value in the index

    option+'s' sets the value you have selected but lowers the case

    'w' pops up a window that allows you to input an index value

    'j' undoes the last set value and returns you to the previous cluster

    the space bar skips a cluster and sends you to the next term
